Metodologia

Servidor

  • Linguagem C/C++
  • Protocolos: RTCP(Real Time Control Protocol)/RTSP(Real Time Stream Protocol)/RTP(Real Time Protocol)*
  • Biblioteca FFmep ou JMpeg (codificação)

Cliente

  • Aplicação Web*
  • Biblioteca FFmep ou JMpeg (decodificação)

Jogo

  • Linguagem C/C++
  • Biblioteca SDL

Diferenças

  • Livre
  • Captura frames diretamente do buffer

Vantagens

  • Não necessita de virtualização
  • Mais Acessível (console)

Desvantagens

  • Alteração no código
